@pipeletteio/nop

@pipeletteio/nop

A simple nop (no-op) function helper. Nop is a function that does nothing.

Installation

 
npm install @pipeletteio/nop

Example

 
import { nop, isNop } from '@pipeletteio/nop';
// const { nop, isNop } = require('@pipeletteio/nop');

isNop(nop); // => true

isNop(function () {}); // => false
